Extract the archive, with paths, to OSB_ORACLE_HOME, on each OSB server. Restart each OSB server after 
deployment.

There should be 3 files added (or updated) on each server.

$OSB_ORACLE_HOME/config/xpath-functions/osb-x-forwarded-for.jar
$OSB_ORACLE_HOME/config/xpath-functions/osb-x-forwarded-for.properties
$OSB_ORACLE_HOME/config/xpath-functions/osb-x-forwarded-for.xml
